# holodos

## Project setup

```
npm i
```

## Project .env

```
TELEGRAM_TOKEN : Токен бота в telegram
WEB_APP_URL    : Адрес web-сайта
PROXY_SERVER   : Прокси-сервер (http://user:password@host:port/)
```

### Development

```
npm run dev
```

### Production

```
npm run start
```
